# 💻Projeto: Meu primeiro portifólio
<h3>Projeto feito com HTML, CSS e JS puro!<h3>

### 🚀Começando
><p>Essas instruções permitirão que você obtenha uma cópia do projeto em operação na sua máquina local para fins de desenvolvimento e teste.</p>

### 📋Pré-requisitos
>Antes de começar, você precisa ter instalado em sua máquina as seguintes ferramentas: [git](https://git-scm.com/)</br>
>Além disto é bom ter um editor para trabalhar com o código como o [VSCode](https://code.visualstudio.com/).

```bash
1. Clone o repositório
  $ gitclone <https://github.com/thayannecarrilho/firstPortifolio.git>

2. Acesse as extensões no VSCode e instale "Live Server"

3. Execute o arquivo HTML com o Live Server

```
